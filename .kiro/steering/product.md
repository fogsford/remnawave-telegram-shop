# Product Overview

Remnawave Telegram Shop is a Telegram bot for selling VPN subscriptions with integration to Remnawave (https://remna.st/). The bot enables users to purchase and manage VPN subscriptions through Telegram with multiple payment options.

## Core Features

- VPN subscription sales with multiple payment methods (bank cards, cryptocurrency, Telegram Stars, Tribute)
- Multiple subscription plans (1, 3, 6, 12 months)
- Trial subscriptions with configurable duration and traffic limits
- Automated subscription management and renewal
- Subscription expiration notifications (3 days before expiry)
- Multi-language support (Russian and English)
- Referral system with configurable rewards
- Selective squad assignment for user management
- Admin commands for user synchronization

## Payment Systems

- YooKassa API (bank cards)
- CryptoPay API (cryptocurrency)
- Telegram Stars (native Telegram payments)
- Tribute (subscription service)

## Key Integrations

- Remnawave VPN panel API
- Telegram Bot API
- PostgreSQL database
- Optional: Moynalog analytics
